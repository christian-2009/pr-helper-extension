GitHub PR Comment Tracker

Never miss an important comment again.

This extension helps you track and jump to comments that still need action in any GitHub PR.

A comment is considered actioned if:

  • It has been marked as resolved
  • It has a reaction
  • The author has replied last in the thread

All other comments are shown as "left to action".

Click a comment in the list to scroll to it instantly on the PR page.
